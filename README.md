# Aevum

> A replay-first, policy-governed context kernel.

Aevum sits between raw data and AI consumers. It ingests data through a governed
membrane, records canonical truth in an append-only episodic ledger, assembles
bounded context through graph traversal, and enables deterministic replay of any
past decision.

## Status

Pre-release. Active development. Not yet suitable for production use.
