# Enterprise Overview

Aevum is designed for regulated and high-stakes deployments. It provides
a cryptographic audit trail, consent-checked data access, human review
gates, and five unconditional barriers — the building blocks for
accountable AI systems in healthcare, finance, legal, and other regulated
contexts.

Before evaluating Aevum for a regulated deployment, read:
- [Threat Model](../../THREAT_MODEL.md) — what Aevum protects, what it
  does not, and trust assumptions
- [Control Mapping](../../CONTROL_MAPPING.md) — Article-by-Article and
  standard-by-standard coverage mapping
