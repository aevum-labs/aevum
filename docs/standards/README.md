# Standards

This directory contains protocol specifications and compliance profiles for Aevum.

| Document | Description |
|----------|-------------|
| [scitt-profile.md](scitt-profile.md) | Aevum SCITT Profile for AI Agent Governance Statements |

These documents describe how Aevum implements or profiles existing IETF and
W3C standards. They are not normative specifications; they describe how Aevum's
implementation relates to each standard.
