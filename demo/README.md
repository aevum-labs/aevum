# Aevum — Live Playground

Interactive API explorer for the five Aevum functions: `ingest`, `query`,
`review`, `commit`, and `replay`.

Every operation runs against the real `aevum-core` kernel. The sigchain
is real. The barriers are real. The GOVERN gate is real.
